# Smart E-Commerce Price Tracker

## Overview
This is a Python-based application that tracks product prices from an e-commerce website. It compares the current price with a target price entered by the user and sends simulated email and SMS notifications when the price drops.

## Features
- Track product prices
- User email input
- User phone number input
- Target price input
- Price history saved in CSV
- Simulated Email Notification
- Simulated SMS Notification

## Technologies Used
- Python
- Requests
- BeautifulSoup
- Pandas

## Installation

```bash
pip install -r requirements.txt
```

## Run the Project

```bash
python etrack.py
```
